# Advanced Lessons Plan

## Lesson 11: Advanced Flexbox & CSS Grid

### Learning Outcomes
- Master advanced flexbox properties (flex-grow, flex-shrink, flex-basis, order)
- Create complex grid layouts with named areas and grid lines
- Understand auto-fit vs auto-fill for responsive grids
- Combine flexbox and grid for sophisticated layouts
- Implement real-world layout patterns

### Key Topics
1. **Advanced Flexbox Deep Dive**
   - The flex shorthand property explained
   - flex-grow, flex-shrink, flex-basis in detail
   - Using order property for visual reordering
   - Flex wrap and alignment tricks
   - Flexbox for card layouts and navigation systems

2. **Advanced CSS Grid**
   - Named grid lines and areas
   - Grid template areas for semantic layouts
   - auto-fit vs auto-fill for responsive design
   - Minmax() function for flexible sizing
   - Grid gap and alignment properties
   - Spanning cells and overlapping content

3. **Combining Grid and Flexbox**
   - When to use Grid (2D layouts)
   - When to use Flexbox (1D layouts)
   - Using Grid for page structure, Flex for components
   - Nested grids and flexbox containers

4. **Real-World Layout Patterns**
   - Holy Grail layout
   - Magazine-style layouts
   - Masonry-style grids
   - Sidebar layouts with flexible content
   - Dashboard layouts

### Worked Examples
- Complete responsive dashboard layout
- Magazine-style article layout with sidebar
- Auto-fitting card grid that adjusts to screen size

## Lesson 12: Component-Based Design & Maintainable CSS

### Learning Outcomes
- Design reusable component patterns
- Implement BEM or similar naming conventions
- Create a mini design system with CSS custom properties
- Build standardized UI components (cards, buttons, forms, headers, footers)
- Organize CSS for maintainability and scalability

### Key Topics
1. **Component Thinking**
   - What is a component?
   - Breaking UI into reusable pieces
   - Component composition and nesting
   - Props-like thinking with CSS classes

2. **CSS Naming Conventions**
   - BEM (Block Element Modifier) methodology
   - Why naming matters for maintainability
   - Creating clear, self-documenting CSS
   - Avoiding specificity wars

3. **CSS Custom Properties (Variables)**
   - Creating a color palette
   - Typography system with variables
   - Spacing scale for consistency
   - Theming with custom properties
   - Dark mode implementation

4. **Building a Component Library**
   - Button variants (primary, secondary, ghost)
   - Card components
   - Form inputs and validation states
   - Navigation components
   - Modal/dialog patterns
   - Alert/notification components

5. **CSS Architecture**
   - File organization strategies
   - Separating base, components, and utilities
   - Creating a scalable folder structure
   - Documentation practices

### Worked Examples
- Complete design system with 10+ reusable components
- Converting existing page to use component-based approach
- Creating theme switcher with CSS variables

## Lesson 13: CSS Animations & Transitions

### Learning Outcomes
- Create smooth property transitions
- Build keyframe animations for complex motion
- Use transform properties effectively
- Implement interactive hover effects
- Build pure CSS interactive components (accordion, tabs, modals)
- Understand animation performance

### Key Topics
1. **CSS Transitions**
   - Transition property syntax
   - Which properties can be animated
   - Timing functions (ease, linear, cubic-bezier)
   - Transition delays and duration
   - Hover effects and state changes

2. **CSS Transforms**
   - translate() for positioning
   - rotate() for spinning elements
   - scale() for zoom effects
   - skew() for distortion
   - 3D transforms (perspective, rotateX, rotateY)
   - Transform origin

3. **Keyframe Animations**
   - @keyframes syntax
   - Animation properties (duration, delay, iteration, direction)
   - Creating complex motion paths
   - Looping vs one-time animations
   - Animation play states

4. **Interactive Patterns Without JavaScript**
   - Checkbox hack for state management
   - :target pseudo-class for tabs and modals
   - Pure CSS accordion with details/summary
   - Hamburger menu with transitions
   - Dropdown menus
   - Image carousels

5. **3D Transforms and Advanced Effects**
   - Perspective and depth
   - Rotating cubes and cards
   - Flip animations
   - Parallax effects
   - Loading spinners

6. **Performance Considerations**
   - GPU-accelerated properties
   - Will-change property
   - Avoiding layout thrashing
   - requestAnimationFrame (brief introduction)

### Worked Examples
- Complete accordion FAQ section (pure CSS)
- Tabbed interface with smooth transitions
- 3D rotating cube with mathematical precision
- Modal dialog system with backdrop
- Animated navigation menu
- Card flip animation
- Loading spinner collection
