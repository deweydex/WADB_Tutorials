How It Works

This modal system uses the :target pseudo-class to show and hide modals based on the URL hash:

  • Each modal has a unique ID (e.g., id="modal-welcome")
  • Clicking "Welcome Modal" changes URL to #modal-welcome
  • The modal with id="modal-welcome" becomes :target
  • CSS .modal:target { display: flex; } shows the modal
  • Clicking close changes URL to #, hiding the modal

Animations: The backdrop transitions from background: rgba(0,0,0,0) to rgba(0,0,0,0.6), creating a fade effect. The dialog transitions from transform: scale(0.7) to scale(1) with opacity: 0 to 1.

Limitations: Pure CSS modals work well for simple use cases but lack some features like focus trapping, Escape key closing, and full ARIA support. For production applications with accessibility requirements, add JavaScript to enhance these modals.
