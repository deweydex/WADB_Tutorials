How the :target Pseudo-Class Works

When you click a link with a hash (like #modal-info), the browser:

  1. Updates the URL to include that hash
  2. Applies the :target pseudo-class to the element with matching ID
  3. CSS changes the element's display from none to flex
  4. The modal becomes visible with a smooth animation!

Tip: Click the backdrop (dark area) or the √ó button to close the modal. Both link to #, which clears the URL hash and hides the modal.

This modal uses a slide-down animation. The content slides in from the top using transform: translateY(-50px) in the keyframes.

Animation Code

The animation is defined like this: 

@keyframes modalSlideDown {
    from {
        opacity: 0;
        transform: translateY(-50px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

Different Modal Types

  • Fade: Simple opacity transition
  • Slide: Slides in from top (this modal)
  • Scale: Zooms in from center (try the warning modal)
